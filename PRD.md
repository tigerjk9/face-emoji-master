# PRD — 표정 대장 챌린지

## 1. 비전
> 빠르고 재밌는 5라운드 표정 게임으로 부스 회전율 + SNS 공유성 동시 확보.

## 2. 사용자
- 초·중·고 학생 (체험 + 결과 공유)
- 부스 운영자 (짧은 회전 시간 = 많은 학생 수용)

## 3. 사용자 시나리오
1. 「게임 시작」 → 카메라 권한 → MediaPipe 모델 로드 (~10초)
2. 3-2-1-GO 카운트
3. 5라운드 (각 5초): 화면 이모지 → 표정 따라하기 → 0~100점 측정
4. 라운드 결과 플래시 (1.7s) → 다음 라운드
5. 총점 + 칭호 + 라운드별 그리드 표시
6. 「다시 도전」 / 「부스 메인」

## 4. 기능 요구사항
- **F1.** MediaPipe Face Mesh 468 landmarks 실시간 처리
- **F2.** 5종 표정 scorer (Smile/MouthOpen/Surprise/Wink/EyesClosed)
- **F3.** 라운드 ring countdown + 실시간 점수 바 + 최고 점수 트래킹
- **F4.** 5단계 칭호 시스템 (총점 기반)
- **F5.** 라운드별 결과 그리드 (이모지 + 점수)

## 5. 비기능 요구사항
- **개인정보 보호** — 영상 외부 전송 0건, 페이지 이탈 시 카메라 자동 해제
- **HTTPS 필수** (카메라 권한)
- **추가 비용 0** — Gemini 등 외부 API 호출 없음
- 단일 `index.html` + Vercel
- prefers-reduced-motion 자동 비활성화

## 6. 디자인 원칙
- 부스 메인과 동일한 글래스모피즘 + 오로라 (rose-amber 그라디언트)
- 점수는 시각화 우선 (ring countdown + 점수 바)
- 결과 화면에 SNS 공유 욕구 자극 요소

## 7. 성공 지표
- 부스 회전율 (1회당 1~2분)
- 다시 도전 비율
- 결과 화면 캡처 공유 (정성 관찰)

## 8. 향후 (Out of scope)
- 결과 화면 SNS 공유 이미지 자동 생성
- 표정 종류 추가 (😘 키스, 😡 화남, 😋 혀 내밀기)
- 난이도 조절 (Easy 5초 / Hard 3초)
- 멀티 모드 (친구와 동시)
